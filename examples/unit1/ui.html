Separate devices

This setup demonstrates using separate GPIO pins for input and output.

One device mapped to both input and output pins.

Shared pin devices

Multiple virtual devices are connected to the same GPIO pin. They behave just like physically wired devices sharing a connection. Buttons can be made "sticky" to simulate the button press time.

Multiple devices on a single pin.

Composite Interface

This unit simulates both input and output behavior. It sends input to one pin (e.g., simulating a button press), and visually reacts to output from another pin (e.g., simulating an LED). Both directions are handled by the same HTML element.

One device per one input and one output pin
